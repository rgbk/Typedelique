**Interactive Typographic Kaleidoscope with Video Masking**

A creative AI teaching tool for ESMOD Paris students exploring the intersection of typography, video, and generative design.

## üöÄ Usage

### Basic Controls
1. Open the app
2. Choose a font from the dropdown
3. Enter a character (letter, number, symbol)
4. Adjust rotation speed and size
5. Upload your own video (optional)

### Advanced Features
- **Clipping**: Enable to contain effects within boundaries
- **Wireframe**: Show visual guides
- **Background Video**: Enable for full-screen video layer
- **Position Slider**: Move kaleidoscope elements toward corners
